How do I define a function in Python and why is it useful?

AI Tutor

Functions are reusable blocks of code that perform a specific task. They help keep your code dry (Don't Repeat Yourself) and organized.

You define them using the def keyword followed by the function name and parentheses.

python 
def greet(name):
    """Simple function to greet a user"""
    return f"Hello, {name}!"
# Calling the function
message = greet("Alex")
print(message)

What about default parameters?

AI Tutor

Default parameters allow you to provide a fallback value if no argument is passed during the function call. This makes your functions more flexible.

python 
def greet(name="Student"):
    return f"Hello, {name}!"
print(greet())          # Output: Hello, Student!
print(greet("Sarah"))   # Output: Hello, Sarah!
